Design System Information

Typography

Primary Font: Pompiere

The Pompiere font family is used consistently throughout this website for all text elements. It provides a unique, readable character while maintaining accessibility standards.

Font Specifications

  • Font Family: "Pompiere", sans-serif
  • Font Weight: 400 (regular)
  • Font Style: normal
  • Source: Google Fonts

Spacing System

This website uses a consistent spacing system based on multiples of 8 pixels to ensure visual harmony and accessibility.

Spacing Scale

  • 8px: Small spacing for tight elements
  • 16px: Standard spacing for most elements
  • 32px: Medium spacing for section separation
  • 64px: Large spacing for major layout divisions

Application Examples

  • Navigation items: 16px padding
  • Section margins: 32px to 64px
  • Text line height: calculated for optimal readability
  • Button padding: 8px to 16px based on size
